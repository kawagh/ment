# ment
## what is this?
ment is a python library to write daily log in markdown quickly and to synthesize daily logs based on category.

## prerequisities
- vim

## how to install and run ment
```
pip install ment
m # start editting ~/ment_dir/<todays_date>/diary.md
```

## structure
